# What is JavaScript?

JavaScript (opens new window) was initially created to serve and fulfill a single purpose – to make webpages feel alive – of course, that goal was achieved and now we're using it for much different things.

Usually programs written in JavaScript are referred to as scripts, they live inside of the HTML's DOM and often run client-side when the page has been loaded. Scripts are provided and executed as plain text. They don’t need special preparation or compilation to run.

Note:

Knowing the differences (opens new window) between compiled and interpreted languages is quite useful!

JavaScript itself is rather flexible and forgiving as a language. It is one of the most weakly-typed (opens new window) languages out there. However, that is not to say that it is a bad language, or any way inferior to other languages. Some of it's main strengths come from the way it is so weakly-typed and versatile – for this reason developers have written a variety (opens new window) of tools built on top of the core JavaScript language, these include:

  • TypeScript (opens new window)
  • Libraries (opens new window)
  • APIs (opens new window)
  • And many, many more...
